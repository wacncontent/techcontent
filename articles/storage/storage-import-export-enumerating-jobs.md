# 枚举作业
若要枚举某个订阅中的所有作业，请调用[列出作业](https://docs.microsoft.com/zh-CN/rest/api/storageimportexport/jobs#Jobs_List)操作。“`List Jobs`”将返回作业列表及以下属性：

-   作业的类型（导入或导出）

-   当前作业状态

-   作业的关联存储帐户
